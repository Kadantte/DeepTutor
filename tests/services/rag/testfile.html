What is deep learning?

Deep learning is a branch of machine learning that uses multi-layer neural networks to learn hierarchical representations of data. It has achieved major breakthroughs in areas such as image recognition, natural language processing, and speech recognition.

Basic components of a neural network

  • Input layer: Receives raw input data
  • Hidden layers: Perform feature extraction and transformation
  • Output layer: Produces the final prediction

Common deep learning models

Model type Main applications Key characteristics
Convolutional Neural Network (CNN) Image classification, object detection Local receptive fields, parameter sharing
Recurrent Neural Network (RNN) Sequence modeling, time-series forecasting Handles variable-length sequences, supports memory
Transformer Natural language processing, large language models Self-attention, parallel computation

The deep learning training process

  1. Data preprocessing: cleaning, normalization, data augmentation
  2. Forward pass: compute predictions
  3. Loss computation: compare predictions with ground truth
  4. Backpropagation: compute gradients
  5. Parameter update: update weights with an optimizer

Common activation functions

Activation functions introduce nonlinearity into neural networks. Common choices include:

  • ReLU: f(x) = max(0, x)
  • Sigmoid: f(x) = 1 / (1 + e^(-x))
  • Tanh: f(x) = (e^x - e^(-x)) / (e^x + e^(-x))
  • Softmax: used in the output layer for multi-class classification
